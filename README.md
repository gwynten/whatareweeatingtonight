# what are we eating tonight?

A mobile-responsive food voting prototype.

## How to publish with GitHub Pages

1. Create a new GitHub repository named `what-are-we-eating-tonight`.
2. Upload `index.html` from this folder.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/root**
5. Click **Save**.
6. Wait 1–3 minutes. Your site will appear at:

`https://YOUR-GITHUB-USERNAME.github.io/what-are-we-eating-tonight/`

## Notes

This version is a frontend prototype. Group votes are saved in browser localStorage. To make it fully live for multiple users, connect it to Firebase or Supabase.
